5. Methodology and Reproducibility Notes

Source universe

The 20-stock universe was selected from docs/analysis results/Watchlist_Returns_Fundamentals_2026-05-29.html. The HTML table was parsed with pandas.read_html, the 30 Day Return column was converted from percent text to a numeric value, and the rows were sorted descending. The top 20 symbols were: MXL, NVTS, WOLF, UMC, 000660.KS, HEXI.ST, ARM, SIMO, ALAB, MU, RKLB, AMD, DOCN, LUNR, POET, ACMR, INTC, SNDK, QCOM, HSYDF.

May high pullback calculation

Daily OHLCV data came from the local QuantCore SQLite cache at data/quantcore.sqlite, table ohlcv, filtered to interval = '1d' and dates from 2026-05-01 through 2026-05-31. The cache had daily bars through 2026-05-28 for all 20 symbols, so the pullback ranking is as of the latest cached close, not a live market refresh.

For each symbol, May High is the maximum daily high in May, Latest Close is the latest cached May daily close, and the pullback metrics are: $ Off High = Latest Close - May High and % Off High = (Latest Close - May High) / May High * 100. The table is ranked from most negative % Off High to least negative.

MCP tools used

The rebound and sink classifications were based on MCP tool outputs for the same 20 symbols. The primary composite tool was mcp__stock_price_server.get_trade_recommendation with capital = 5000. It synthesized RSI, MACD, stochastic oscillator, Bollinger Bands, volume analysis, candlestick patterns, dark-pool proxy, short interest, bid/ask spread, unusual call activity, delta-adjusted open interest, options positioning, stop-loss analysis, and news sentiment where available.

Additional confirmation tools were run on the clearest rebound and downside candidates: get_vwap, get_higher_lows, and get_obv. These were used to distinguish real rebound structure from names that were merely extended or experiencing a short-term bounce inside a weaker setup.

The gamma-wall overlay used mcp__stock_price_server.get_delta_adjusted_oi with max_expirations = 3. The report records each symbol's current MCP price, gamma wall strike, and market-maker hedge bias. Symbols without usable options data were marked as N/A rather than inferred.

Interpretation rules

  • Rebound candidates required a positive MCP composite read plus supporting evidence such as narrowing spreads, bullish call activity, positive news, VWAP support or reclaim, higher-low structure, and rising OBV.
  • Likely sink or avoid-long names had negative or conflicted MCP composites plus bearish momentum/candle signals, overbought exhaustion, weak or negative news, spread stress, deteriorating fundamentals, or no higher-low support.
  • Mixed/watch names had bullish components but poor entry quality, overbought momentum, weak risk/reward, or conflicting technical and sentiment signals.
  • Gamma interpretation treated sell_on_rally as a rally-resistance warning. A gamma wall below current price was interpreted as possible downside magnet/support, while a nearby wall above current price was interpreted as possible pin/resistance rather than automatic upside confirmation.

Important limitations

This report is based on local cached OHLCV data and MCP tool snapshots available at creation time. The May high table does not include a live refresh beyond the cached 2026-05-28 close. MCP outputs can change as prices, options open interest, spreads, news, and cached fundamentals update. To recreate the analysis exactly, use the same source report, the same local QuantCore cache state, and the same MCP tool parameters listed above.
